# Online School Startup MVP

Demo MVP онлайн-школы в mobile-first стиле.

## Что внутри
- вход по email + одноразовый код (демо)
- роли: ученик / преподаватель / администратор
- экран подписки
- курсы → модули → уроки → тест
- логика прохождения урока только после успешного теста
- пересдача теста
- прогресс ученика
- кабинет преподавателя
- кабинет администратора

## Запуск
```bash
npm install
npm run dev
```

Открыть:
```bash
http://localhost:5173
```
